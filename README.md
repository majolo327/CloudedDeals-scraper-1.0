# CloudedDeals-scraper-1.0

A deal-tracking platform with a Playwright scraper, Next.js frontend, and Supabase backend.

## Project Structure

```
clouded-deals/
├── scraper/           # Python Playwright scraper
├── frontend/          # Next.js 14 React frontend (TypeScript + Tailwind)
├── supabase/          # Database migrations
└── docs/              # Documentation
```

See [docs/README.md](clouded-deals/docs/README.md) for setup instructions.
